  • Protocolo automático
  • Cálculo de prazos
  • Email automático

Sistema de Anexos

Upload e download seguro de documentos.

  • Validação de tipo MIME
  • Limite de 10MB
  • Controle de acesso
  • API REST completa

Notificações Email

6 tipos de notificações automáticas.

  • SMTP configurável
  • Templates HTML
  • Cron automático
  • Alertas de prazo

Dashboard Admin

Painel completo para gestão.

  • 6 cards de estatísticas
  • Filtros avançados
  • Responder pedidos
  • Tramitação

⚡ Scripts de Automação

deploy.sh

Script completo de deploy automatizado (500+ linhas)

Funcionalidades:

  • Detecta distribuição Linux
  • Instala todas as dependências
  • Configura banco com senha gerada
  • Instala SSL (Let's Encrypt)
  • Configura firewall e Fail2Ban
sudo ./deploy.sh
comandos-rapidos.sh

Menu interativo com 11 comandos úteis

Opções:

  • Ver status de serviços
  • Logs em tempo real
  • Backup/Restaurar
  • Renovar SSL
  • Ver estatísticas
sudo bash comandos-rapidos.sh
