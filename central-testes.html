📋 Como Usar Esta Central:

  1. Clique em cada botão - Eles abrem em novas abas do navegador
  2. Teste em ordem: Comece pelos "Testes Básicos" (1, 2, 3)
  3. Se aparecer página branca:
    • Pressione F12 (Ferramentas do Desenvolvedor)
    • Vá na aba "Console" - Veja erros JavaScript
    • Vá na aba "Network" - Veja se arquivos carregam
    • Vá na aba "Application" - Limpe cache/cookies
  4. Limpar cache: Ctrl + Shift + Delete
  5. Modo anônimo: Ctrl + Shift + N (Chrome) ou Ctrl + Shift + P (Firefox)
  6. Testar em outro navegador: Chrome, Firefox, Edge

💡 Dicas de Troubleshooting

  • ✅ Se teste1.php funciona → PHP está OK
  • ✅ Se teste2.html funciona → Apache está OK
  • ✅ Se teste3.php redireciona → Redirecionamento OK
  • ❌ Se tudo fica branco → Problema de cache ou JavaScript
  • ❌ Se só algumas páginas falham → Problema específico do arquivo
